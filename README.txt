This directory contains all of the data files required by egoboo.

LINUX

You should not run the Makefile in this directory, but instead you should run the Makefile in the egoboo sourcecode directory.

WINDOWS/MAC/OTHER
No installation file or installer are provided. 

Take the following steps:

- make a directory where you want the program to run
- copy the executable to this directory
- copy the contents of this direcotry to the the same directory
- if you did not put the required dlls in the search path when you built egoboo, copy these dlls into the same directory, as well

Launching the executible from this directory will start the game.
