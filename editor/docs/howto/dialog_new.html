EGOBOO MAP EDITOR: How-To V 0.1.5


Creating new map
As a first step you have to give at minimum the map size and click 'Ok'.
The map size can be changed using the arrow keys.
The other data can at any moment be changed over the menu point 'Tools' -> 'Main-Info';



Drawing the map
Next you have to choose 'Map' from the tools menu. This allows you to draw the map in a simple mode:

  • Right Mouse button pressed: Carves out the solid map and sets floor tiles
  • Left Mouse button pressed: Creates a wall tile
A single click sets a single tile, with the mouse dragged with pressed button multiple tiles can be chosen. The tiles are set as soon as the mouse button is released.
