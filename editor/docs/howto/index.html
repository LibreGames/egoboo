EGOBOO MAP EDITOR: How-To V 0.1.7


General info
- At the moment the maximum map size supported is 128 x 128 tiles
- Maps are loaded form the directory 'module' and must have the name 'level.mpd'
- Maps are saved to the directory given by the config-file and and are named 'level.mpd'
- Choosing and editing tiles is done in the 2D-Window, the changes are shown in the 3D-Window
- For the map geometry, the data for the fan type is 'hard-coded' as 'preset'
- Tile-Size is 128 Units
- Inital map size can be set in the CONFIG-File with the value "mapsize"

Not all functions are implemented as discribed yet


Directories
- 'data': Holds general data used by editor
- 'docs': Holds all documentation for the editor
- [module/... ]: Holds the data of the module actually worked on. To be copied in '[module.obj]/gamedat' directory
- 'src': Holds the complete source code of the editor

General Keyboard-Commands

  • ESC quits program
  • LEFT-CONTROL-m: Shows/hides 2D-Map, if a map is loaded

Camera movement commands

  • Keypad 8: Move forward
  • Keypad 6: Move backward
  • Keypad 4: Move left
  • Keypad 6: Move right
  • Keypad 7: Turn left
  • Keypad 9: Turn right
  • Keypad+: Zoom in
  • Keypad-: Zoom out
  • Arrow Up: Move ahead
  • Arrow Down: Move backward
  • Arrow Left: Move left
  • Arrow Right: Move Right
Pressed Left-Shift Key lets the camera move with double speed


File Menu

  • 'Load..': Loads map and other info from the 'gamedat' directory from 'workdir' given in the 'editor.cfg'-File
  • 'Save': Saves the actual loaded data for the level
  • 'New...': Opens an empty dialog with the basic data ('menu.txt') for creation of a new map.
  • 'Exit': Quits the editor (as keyboard command ESC)



Settings Menu
Here you can switch on/off different modes how the map is drawn



Tools Menu
Here you chose the edit mode. The chosen edit state is displayed at the bottom of the window.
Based on the edit mode, mouse clicks into the 2D-Map are interpreted differently.

Passages are displayed blue on map
Spawn points are displayed red on map
A newly chosen area is displayed blue on map



Working with tools
First the tool is chosen in the menu and then the tool is applied on the tile(s) chosen by the user.
Following ist he description of the tools

None / View
In this mode, nothing can be changed on map, only infos about the map are displayed with a right click on a map tile (not implemented yet).

Build map
An area chosen with the right mouse button pressed creates floor tiles.
An area chosen with the left mouse button pressed creates walls.
Adjacent tiles are adjusted accordingly.

Single Tile
Changes a single chosen tile to a new tile chosen with the Fan-Dialog (not implemented yet)

Single Tile
Changes a single chosen tile to a new tile chosen with the Fan-Dialog (not implemented yet)

Tile-FX
After an area is chosen, a dialog is opened with checkboxes for the different flags.
If the 'OK'-Button is pressed, all flags of the chosen tiles are changed to the flags given in the dialog box.

Tile-Texture
After an area is chosen, a dialog is opened for chossing a different texture for the chosen tiles. If the 'OK'-Button is pressed, the texture of the chosen tiles is changed to the new one chosen.

Passage
If an existing passage is chosen by a single tile clicked on, then a dialog is opened with the info about the existiong passage for editing (except size of area)
If an area is chosen, then a new passage is created for given area and a dialog is opened for editing of the properties of the passage.

Spawn point
If an existing spawn point is chosen, then a dialog is opened with the info about the existing spawn point for editing.
If there is now spawn point yet, a new one is created and a dialog opened to set the properties of the spawn point.


Technical info
Fan-Commands used for 'simple' mode:
-  0: Floor
-  1: Top, impassable
-  8: Wall, looking north
- 16: Outer edge north-east
- 19: Inner edge north-west
