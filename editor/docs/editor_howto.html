EGOBOO MAP EDITOR: How-To


General info
- At the moment the maximum map size supported is 128 x 128 tiles
- Maps are loaded form the directory 'module' and must have the name 'level.mpd'
- Maps are saved to the directory 'module' and and are named 'level.mpd'
- Choosing and changing tiles is done in the 2D-Window, the changes are shown in the 3D-Window
- For the map geometry, the data for the fan type is 'hard-coded' as 'preset'
- Tile-Size is 128 Units

General Keyboard-Commands
- ESC quits program
- LEFT-CONTROL-m: Shows/hides 2D-Map, if a map is loaded
- 'I' : Toggles Display of information about chosen fan
- 'B': Toggles 'Browsing' trough all fan shapes (displayed at chosen position) Browse with arrow keys.
Planned commands (if a tile is chosen in map):
- 'R': 'R'otate chosen fan shape
- 'S': Toggle 'S'olid mode
- 2D Map: Mouse Right-Click: Set Floor tile / Mouse left-Click: Set Wall Tile

Camera movement commands
Keypad 8: Move ahead
Keypad 6: Move backward
Keypad 4: Move left
Keypad 6: Move right
Keypad 7: Turn left
Keypad 9: Turn right
Keypad+: Zoom in
Keypad-: Zoom out

Directories
- 'data': Holds general data used by editor
- 'docs': Holds all documentation for the editor
- 'module': Holds the data of the module actually worked on. To be copied in '[module.obj]/gamedat' directory
- 'src': Holds the complete source code of the editor

Technical info
Fan-Commands used for 'simple' mode:
-  0: Floor
-  1: Top, impassable
-  8: Wall, looking north
- 16: Outer edge north-east
- 19: Inner edge north-west

The vertices are 'rotated' in other direction, if needed
The 'tx_flags' of the fan are used to sign for the editor, which way a wall is oriented:
0: North
1: East
2: South
3: West
