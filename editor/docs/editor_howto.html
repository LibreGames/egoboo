EGOBOO MAP EDITOR: How-To


General info
- At the moment the maximum map size supported is 128 x 128 tiles
- Maps are loaded form the directory 'module' and must have the name 'level.mpd'
- Maps are saved to the directory 'module' and and are named 'level.mpd'
- Choosing and changing tiles is done in the 2D-Window, the changes are shown in the 3D-Window
- For the map geometry, the data for the fan type is 'hard-coded' as 'preset'
- Tile-Size is 128 Units
- Map size for new maps is read from the configuration file
- Inital map size can be set in the CONFIG-File with the value "mapsize"
- Tile flags and texture data can be changed using the 'Update' button in the fan info dialog
- 'Simple' editing needs test of functionality

General Keyboard-Commands
- ESC quits program
- LEFT-CONTROL-m: Shows/hides 2D-Map, if a map is loaded
- 2D Map: Mouse Right-Click: Set Floor tile / Mouse left-Click: Set Wall Tile

Editing
For editing, the dialog for the tile properties has to be open. Here it can be switched between the different edit modes.
The following edit modes are available.

Fan Info (No Edit)
If a tile is chosen in the 2D-Map, the info about the tile with flags and textures are displayed.
It works only in 'single-tile-mode'.

Edit Simple Mode
- Click/drag with right mouse button: Changes all chosen tiles to bottom tiles
- Click/drag with left mouse button: Changes all chosen tiles to wall tiles
The adjacent tiles are adjusted as well as possible

Edit Single Fan
- Click with right mosue button: Choose tile
- Click with left mouse button: The chosen tile is set to the tile type with flags and texture as displayed in the properties dialog

Commands for preparing / choosing tiles in free mode:
SPACE-Key: Browses troug the five possible different tile types to set
R-Key: Rotates the chosen tile type as needed

Edit Fan-FX
- Click/drag with right mouse button: Set all chosen tiles to FX displayed in properties dialog

Edit Fan-Texture
- Click/drag with right mouse button: Set all chosen tiles to Texture displayed in properties dialog

Camera movement commands
Keypad 8: Move ahead
Keypad 6: Move backward
Keypad 4: Move left
Keypad 6: Move right
Keypad 7: Turn left
Keypad 9: Turn right
Keypad+: Zoom in
Keypad-: Zoom out
Arrow Up: Move ahead
Arrow Down: Move backward
Arrow Left: Move left
Arrow Right: Move Right

Directories
- 'data': Holds general data used by editor
- 'docs': Holds all documentation for the editor
- 'module': Holds the data of the module actually worked on. To be copied in '[module.obj]/gamedat' directory
- 'src': Holds the complete source code of the editor

Technical info
Fan-Commands used for 'simple' mode:
-  0: Floor
-  1: Top, impassable
-  8: Wall, looking north
- 16: Outer edge north-east
- 19: Inner edge north-west

The vertices are 'rotated' in another direction, if needed
The 'tx_flags' of the fan are used to sign for the editor, which way a wall is oriented:
0: North
1: East
2: South
3: West
