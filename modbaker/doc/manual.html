Manual

Controls

Key Description
Arrow keys Move the camera
Page up / down Zoom in / out
Esc Quit ModBaker
W increase the height of the selected vertices
S Decrease the height of the selected vertices
Space Clear the selection
F10 Save the .mpd file
Shift Move faster
When holding shift while clicking, the vertex gets added to the selection.

Module loading

Copy a module into the modules directory.
Then use the following command to load the module:

./modbaker "zippy.mod"

Or for Windows:
modbaker.exe "zippy.mod"

Building ModBaker

To build ModBaker, use the Makefile.

Required libraries

  • libSDL
  • SDL_TTF
  • SDL_image
  • OpenGL
